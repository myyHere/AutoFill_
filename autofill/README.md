# 求职官网自动填表（小白版）

这是一个 Chrome 插件，不需要写代码。

## 功能
- 一次录入个人信息
- 在招聘官网页面一键自动填写
- 支持两种模式：
  - 本地模式：不调用 AI，免费稳定
  - AI 模式：调用大模型理解页面字段，命中率更高
- 支持导入/导出配置，方便多人推广

## 1 分钟安装
1. 打开 Chrome，进入 `chrome://extensions/`
2. 右上角打开“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录：`2025study/autofill`
5. 浏览器右上角固定该插件

## 使用步骤
1. 打开插件，填入你的基本信息
2. 选择填写模式：`本地模式` 或 `AI 模式`
3. 如果用 AI 模式，填写：
   - `AI API Key`
   - `AI Base URL`（默认 `https://api.openai.com/v1`）
   - `AI 模型`（默认 `gpt-4.1-mini`）
4. 可点击“验证 AI API”检查 Key 和接口是否可用（也可点“保存并验证”）
5. 点击“保存信息”
6. 打开目标招聘官网填写页面
7. 点击“一键填充当前页”

## 快速导入你的模板
- 目录里已提供：`profile-template.json`
- 在插件中点击“导入配置”，选择该文件即可（模板默认全空白）
- 按需填写你的个人信息并保存

## 导入导出（推广友好）
- 导出：点击“导出配置”，得到 `autofill-profile.json`
- 导入：点“导入配置”选择 JSON 即可

## 注意
- 不同官网字段命名差异较大，个别项可能识别不到，需要手动补一下
- 当前版本支持字段：姓名、手机、邮箱、GitHub、学校、在校时间、学院、专业、学校城市、家庭城市、GPA、专业排名、实验室、奖学金/荣誉、项目名称/项目时间/项目内容、竞赛获奖/获奖时间/获奖内容，以及通用字段（微信、城市、学历、工作年限、当前公司/职位、期望薪资、作品链接、自我介绍）
- AI 模式会把“页面字段描述 + 你的资料”发送给你配置的模型服务，请自行确认隐私与合规
- 信息保存在浏览器扩展存储中，不会自动上传到其他服务器（AI 模式调用的接口除外）
